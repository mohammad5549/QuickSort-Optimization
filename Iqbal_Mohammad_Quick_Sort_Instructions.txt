Instructions for running the Quick Sort MIPS program:

1: Open program using MARS
2: Click assemble
3: Click run
4: Observe Sorted Array

Instructions for running the Bubble Sort C++ program:

1: Compile the program using the command: "g++ -g Quick_Sort.cpp -o main"
2: Run the program using the command "./main"
3: Observe elapsed times

Instructions for running the Quick Sort Optimized Assembly program:

1: Create object file using the command: "as -o Quick_Sort.o Quick_Sort.s"
1: Compile the program using the command: "g++ -o main Quick_Sort.o"
2: Run the program using the command "./main"
3: Observe elapsed times
